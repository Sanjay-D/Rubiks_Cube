# Rubiks_Cube
A simple Rubik's Cube simulator using C++  
HOW TO PLAY :  
1. Move your mouse to the side of the screen to turn the cube  
2. Use: w,a,s,d,x to move sides  
	w-Up  
	a-Left  
	s-Front  
	d-Right  
	x-Down  
	  
	Use Shift+"Letter" to make the oppsite move  
3. Hit 'l' to shuffle the cube
